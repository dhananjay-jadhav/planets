# Planets Project

This project parses Kepler data to find habitable planets.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dhananjay-jadhav/planets.git
   ```
2. Install dependencies:
   ```bash
   yarn install
   ```

## Usage

To run the project, use the following command:

```bash
yarn dev
```

This will execute the script to parse the Kepler data and log the habitable planets found.

## Available Scripts

- `yarn build`: Compiles the TypeScript code.
- `yarn dev`: Runs the main script using `ts-node`.
- `yarn test`: Runs the tests using Jest.
